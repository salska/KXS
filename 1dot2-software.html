Software

Several software tools are required to implement a full end to end knowledge management solution.

Zotero

Citations are used to accurately identify written sources including books, papers and website references.

DevonThink

All assets are stored in digital form on disk and are indexed and tagged for retrieval and annotation using DT3.

Obsidian

Knowledge management provides the platform for binding together citations and digital assets with user entered commentary in a connected way that allows concepts and ideas to be linked in a fluid way.
