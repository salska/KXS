Training Videos

When watching a training video or or online lecture it is helpful to make notes of any key learnings and extend the knowledge and practice into other areas of work or projects.

The issue is that video materials are spread across youtube and personal stores. The links to youtube videos can be managed by adding a url link to the video withinZK. Also, time markers can be used to highlight specific entry points into where the material can be located.

Where materials are stored locally, these can be indexed usingDT and a link can be made. TheZK code can be assigned to the video asset inDT. This is a manual step but it does mean thatDT can now be used to link together all tags.

How to make effective use of System

  1. Index a folder of books ortrainingvideo withinDT
  2. The index will appear as a database
  3. Remove all auto assigned tags
  4. Tag allbook andtrainingvideo with the general class
  5. As books and videos are watched update the tags to be more specific.
  6. For example:technicalcomedyjava etc. It is better to use simple tags and use appropriate filtering to narrow down when searching.
  7. Add theZKuid to the book or video as a tag
